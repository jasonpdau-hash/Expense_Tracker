# React + Vite

Built off the an existing temple using npm create vite@latest

# Projecct
Expense tracker

- User can add an expense item to a logbook with a title, category, amount, date and description.
- User can modify or delete an expense item

/node_modules - Contains project dependencies
/public - Contains static files served to the browser directly
/src - Contains the applications source code
  main.jsx - Entry point of the app, it renders the root component to the DOM
  app.jsx - The main root component for the app, acts as the central hub.
  index.css - Provides formatting and styling for the app
package.json - Project metadata and dependencies
